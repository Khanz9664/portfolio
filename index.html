House Price Prediction

A predictive model to estimate house prices based on various features of properties using the Ames Housing dataset. (Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn).

View on GitHub

The House Price Prediction project aims to build a machine learning model that predicts house prices based on various features such as the size of the house, number of rooms, neighborhood, and more, using the Ames Housing dataset. This dataset, which includes 79 explanatory variables describing the characteristics of residential homes, serves as the foundation for training the model. The objective is to develop a robust, accurate model that can predict house prices in Ames, Iowa, and potentially be adapted to other real estate markets. (Python, Pandas, NumPy, Scikit-learn, Matplotlib).

This project involves performing Exploratory Data Analysis (EDA) and data visualization on the MovieLens dataset, which contains information about movies, ratings, genres, and users.
